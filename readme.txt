== Description ==

This plugin is a BETA version. Developed to manage exclusive virtual store plans using the WordPress CMS together with the WooCommerce plugin. WordPress: https://br.wordpress.org/ - WooCommerce: https://woocommerce.com/


= Recursos =
* Disponibiliza campo para o cliente informar o valor de troco que precisa (para o entregador levar).
* A informação de troco, tipo de cartão ou cheque fica impresso na Notas do Pedido.
* Aceita Entregas Nativas do WooCommerce (Retirar no local, Taxa fixa e Frete grátis).
* Aceita Correios Brasil (WooCommerce Correios).
* Aceita Entregas Avançadas (Woocommerce Advanced Shipping).
* O sistema aceita todos os pagamentos ou somente um.

== Installation ==

= Instalação =
1. Upload the plugin files to the wp-content/plugins folder, or install using the WordPress plugin installer.
2. Activate the plugin.
3. Go to plugin panel to make basic settings..
